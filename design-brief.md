### 2.4. Блок «Товары» (3 карточки: CutMax / AquaMax / AquaLite)

- **Заголовок блока:** существующий «Серии СОЖ под любую производственную задачу» — оставить или сократить на ваше усмотрение.

#### Состав карточки (свёрнутое состояние)

  - **Изображение бочки** — каждая карточка со своей бочкой (3 разных), на бочке подпись с названием серии. Изображения предоставит заказчик; пока — заглушка.
  - **Название серии** (CutMax / AquaMax / AquaLite) — крупным шрифтом, акцентным цветом.
  - **Тип СОЖ** — короткой строкой под названием:
    - CutMax — «Масляные»
    - AquaMax — «Полусинтетика»
    - AquaLite — «Минералка»
  - **2–3 ключевые характеристики** — короткими bullet-подписями (что важно при первичном выборе). Конкретные пункты предоставит заказчик; пока — placeholder из 2–3 строк.
  - **Две кнопки внизу карточки:**
    - **«Оставить заявку»** — основная, акцентного красного цвета. При клике открывается **Popup-3** «Заявка по {название серии}».
    - **«Подробнее»** — вторичная, нейтральная (outline / прозрачная с обводкой / текстовая со стрелкой ↓). При клике **раскрывает доп. характеристики прямо в карточке** (без перехода на отдельную страницу и без модального окна).

#### Раскрытие «Подробнее» (развёрнутое состояние) — РАСШИРЕННОЕ

При клике на «Подробнее» карточка плавно увеличивается по высоте. Внутри 4 структурированных раздела (с убранным «Заменяет» — он в Приоритете 2):

1. **Применимость** — два мини-блока с иконками-чекбоксами:
   - «По каким операциям»: точение / фрезерование / сверление / шлифование / нарезание резьбы / MQL
   - «По каким материалам»: углеродка / нержавейка / титан / алюминий / чугун / медь-латунь / жаропрочные

2. **Технические характеристики (TDS)** — компактная таблица:
   - Плотность (г/см³)
   - Кинематическая вязкость (мм²/с)
   - pH рабочего раствора
   - Концентрация рабочего раствора (%)
   - Температура хранения (°C)
   - Жёсткость воды для разведения (мг-экв/л)

3. **«Не содержит»** — иконки-бейджи в ряд: без хлора / без серы / без фосфора / без бора / без силикона / без нитритов / без формальдегида (показываем только применимое для каждой серии).

4. **Фасовка** — иконки: 200 л (бочка) / 1000 л (куб) / другие.

Кнопка «Подробнее» меняется на **«Свернуть ▲»**. Кнопка «Оставить заявку» **остаётся видна**.

> Контент 4 разделов (применимость, TDS-цифры, маркеры, фасовка) пришлёт Колян. До этого — плейсхолдеры.

#### Выделение AquaMax как хита продаж

  - **Бейдж «ХИТ ПРОДАЖ»** в правом верхнем углу карточки AquaMax (красная плашка, или жёлтый акцент — на усмотрение дизайнера).

  - **Слегка увеличенный размер** карточки (на ~5–10% по ширине/высоте), либо более контрастный фон/обводка.
  - Возможные дополнительные акценты — на усмотрение дизайнера: тень посильнее, обводка цвета акцента, иной фон карточки, иконка пламени/звезды у названия.

#### Технические уточнения (от заказчика)

- **Артикулы товаров (CM-EP-620, AM-SS-68 и т.д.) на карточках не показываем.** Только название серии + тип + характеристики.

#### От заказчика (Колян пришлёт)
