## TL;DR за 60 секунд

Целевая аудитория TMG — **главные технологи и инженеры-технологи по механообработке** (НЕ «технологи СОЖ» — такой роли почти нет) на средних и крупных металлообрабатывающих предприятиях, 35–55 лет, мужчины. Зарплатный коридор **100–300 тыс. ₽** (СПб, регионы — ниже). Они после ухода Castrol/Shell/Mobil в 2022–2023 живут в режиме «терпим компромисс», но **FUCHS легально работает в Калуге**, Blaser/Quaker идут через дистрибьюторов — премиум-ниша освободилась НЕ полностью.

Главная боль — **«не могу полагаться на поставщика»**: партии СОЖ разные, поставщик может пропасть, на сложных операциях универсалки не тянут. Боль реальна и **подтверждена цитатами с chipmaker.ru**.

Ключевые конкуренты в Питере: **РУСМА, AQUACOOL/СЗКСМ, ЛИККОР** (последний — эталон по маркетингу: открытые цены, портфель Газпром/Северсталь/ОАК на главной, 15+ скачиваемых документов). Реалистичный таргет TMG в 5-летнем горизонте — **200–400 млн ₽ выручки**.

Главная задача маркетинга TMG — **снизить страх перехода** через короткий пилот, инженерное сопровождение и готовый «арсенал материалов», который технолог сможет принести руководству. Главный аргумент — **не «российское производство»** (это уже дефолт), а **«мы решаем биостойкость + пенообразование + специализацию под материал»** — две первые слабости рынка официально признаны индустрией.

## Топ-7 главных инсайтов

### 1. Покупает не один человек — покупают двое, и они разные

**Главный технолог** инициирует, **закупщик** закрывает. Если работать только с одним — сделка сорвётся. Под каждого нужен **свой контент**:
- Для **технолога:** специализация под материал/операцию, инженерия, протоколы испытаний, цифры стойкости инструмента, бесплатная проба.
- Для **закупщика:** документы (УПД, СФ, паспорт партии), отсрочка 30 дней, фасовка, надёжность поставщика, карточка контрагента.

### 2. Главная боль ЦА — НЕ «нужна СОЖ», а «не могу полагаться на поставщика»

Технолог не покупает «лучшую СОЖ». Он покупает **возвращение спокойствия и предсказуемости**. Конкретные подтверждённые боли (с chipmaker.ru):
- *«начала дико вонять»* — биостойкость
- *«Жижа вылазит из бака, вычерпывать пену из бачка»* — пенообразование
- *«дохлые грибы как белесые сопли»* — микробиология
- *«заводская краска на станке начала вспучиваться»* — коррозия
- *«при попадании на руки шкура с рук сходить начинает»* — кожные раздражения
- *«масса производителей колотят СОЖ из отходов производства»* — нестабильность партий

**Что делать:** позиционироваться **не как «лучшая СОЖ», а как «надёжный системный партнёр»**. Тон — спокойный, инженерный, без пафоса. Главный месседж — не «купите наш продукт», а «вернём контроль над участком СОЖ».

### 3. ЦА называет себя НЕ «технолог СОЖ»

🚨 **Открытие из второй волны валидации (hh.ru):** должность «технолог СОЖ» в вакансиях почти не существует. ЦА называет себя:
- «Главный технолог»
- «Инженер-технолог по механообработке»
- «Инженер ОТК»
- «Руководитель отдела снабжения»

СОЖ — лишь **одна из его зон ответственности**. Все коммуникации TMG (тексты сайта, скрипты, SEO) должны строиться вокруг этих ролей, а не «специалист по СОЖ».

### 4. Системные слабости рынка СОЖ официально признаны — это наша ниша

Журнал «РИТМ машиностроения» констатирует:
- «Недостаток отечественных биоцидов хорошего качества»
- «Повышенное пенообразование отечественных эмульгаторов»
- «Российский рынок СОЖ активно заполняют представители Азии»

Это **главные входные ворота для TMG**: биостойкость + пенообразование + специализация под материал. Если у TMG в этом реально есть преимущество — **это месседж №1** на сайте и в коммуникациях.

### 5. Главные страхи технолога — типовые, но критичные

- **«Облажаюсь публично, спросят с меня»** — главный страх. Никого не уволили за то, что купил Лукойл; за TMG — могут спросить лично. Технологу нужно **«покрытие»**: протоколы испытаний, сравнительные таблицы, TCO-расчёт, презентация для главного инженера.
- **«Поставщик исчезнет»** — после 2022 особенно остро. Снижается через прозрачную карточку контрагента, реальную команду, производственную базу.
- **«Меня обманут, как 10 раз до этого»** — *«колотят из отходов»*. Снижается через бесплатный пилот с замерами «до/после», открытость документов.

### 6. TMG как стартап — главный системный риск, который весит больше продукта

«Кто вы такие?» / «А вдруг через год закроетесь?» — топ-2 возражения каждой первой встречи.

**Что делать:**
1. **«Карточка компании» — приоритет №1 на сайте** — производство (фото/видео), команда (имена, опыт, ВУЗы — даже если не критично, это маркер «свои»), реквизиты, бухотчётность.
2. **Накопить первые 3–5 референсов любой ценой** — каждый становится «нитью» для следующих 10 клиентов.
3. **Снижать риск решения** — пилот 200–500 литров, контракт на квартал, гарантия возврата при браке.
4. **Опционально — партнёрство с известным дистрибьютором** для крупных предприятий, которые не пропускают через СБ.

### 7. Каналы продвижения — НЕ те, что мы думали

🚨 **Открытие из второй волны валидации:**
- ❌ **YouTube** — НЕ канал для B2B-СОЖ. Там любители-станочники, не главные технологи.
- ❌ **Telegram-каналы металлообработки** — мало подписчиков (сотни-тысячи), СОЖ почти не обсуждают.
- ❌ **ВК-сообщества** — нет крупных профильных.

✅ **Что реально работает:**
- **chipmaker.ru** — главный профессиональный форум, где есть наша ЦА
- **Тендерные площадки** (rostender.info, b2b-center.ru, zakupki.gov.ru) — реальные лоты на СОЖ есть
- **Точечный SEO** (типа блога Ликкор) — особенно по «СОЖ для алюминия / нержавейки / титана» и «замена Houghton/Castrol/Blaser»
- **Отраслевые выставки** (Металлообработка, MITEX)
- **Холодные продажи** через личные контакты + email на закупщиков
- **Email-рассылки** по списку 200–500 предприятий-целей

## Готовые рекомендации по применению

### Для сайта (приоритет №1, минимальный патч)

1. **Контакты — добавить срочно** (телефон 8-800 или СПб + email + юр-адрес).
2. **Раздел «Продукция»** — три карточки CutMax / AquaMax / AquaLite (название, тип, для какого материала, фасовка, ориентировочная цена).
3. **CTA «Получить пробу 5 л бесплатно»** вместо «Скачать каталог» (если PDF не готов).
4. **Страница «Документы»** — ТУ + паспорт безопасности на каждую линейку, скачиваемые PDF.
5. **Страница «Чем заменить»** — таблица аналогов TMG ↔ Houghton/Quaker/Blaser/Fuchs/Castrol. Это **SEO-магнит**, никто такого не делает.
6. **Блок «Как мы работаем»** — 5 шагов от заявки до сервиса.

После этих 6 пунктов сайт TMG обходит **8 из 13 конкурентов** по полноте.

### Hero/УТП — что писать

- ✅ Использовать: «Российское производство», «Под ваш материал и операцию», «Инженерный подбор рецептуры», «Биостойкость», «Низкое пенообразование», «Замена ушедших брендов».
- ❌ Избегать: «Уникальный», «Революционный», «Лучшие в России», «№1», «Идеальное качество», «100% гарантия».

### Скрипты продаж — главное

**Первая фраза:**
> «Здравствуйте, Сергей Викторович. Иван из TMG, российский производитель СОЖ из СПб. Видел, у вас в основном обработка нержавейки. Звоню узнать — на каких операциях текущая СОЖ не оптимальна. Минут 7 удобно поговорить?»

**Возражение «мы уже работаем с N»** — сразу уточнять **с какой именно маркой**:
- FUCHS → у клиента импорт легально работает, аргумент «российская замена» слабее
- Castrol/Shell/Mobil → острая боль, нужна замена сейчас
- Quaker/Blaser → подсветить нестабильность параллельного импорта
- Лукойл/Газпромнефть → подсветить специализацию vs универсалку

**Возражение «А вдруг закроетесь?»** — давать факты: команда, производство, реквизиты, отчётность; снижать риск через короткий пилот.

### Контент и SEO

**Темы статей под боли (подтверждённые форумом):**
- «Почему пенится СОЖ — 5 причин и решений»
- «Как побороть плесень в баке — практическое руководство»
- «Замена Castrol Hysol / Houghton Hocut — что выбрать в 2026»
- «СОЖ для нержавейки 12Х18Н10Т — на что смотреть»
- «MQL-обработка — какая СОЖ подходит»
- «TCO СОЖ — как считать стоимость на тонну металла»
- «BTA-сверление — особенности подбора СОЖ»
- «Утилизация отработанной СОЖ — как организовать»

**Главные SEO-запросы (после верификации):**
- Коммерческие: «купить СОЖ оптом», «СОЖ оптом производитель», «российские производители СОЖ»
- По материалам: «СОЖ для нержавейки», «СОЖ для титана», **«СОЖ для алюминия»** (был пропущен)
- Замена: «замена Houghton Hocut», «замена Castrol Hysol», «аналог Quaker», «аналог BECHEM»
- Информационные: «почему пенится СОЖ», «биостабильная СОЖ», «подбор СОЖ», «концентрация СОЖ»

### Каналы (приоритет тестирования)

1. **Яндекс.Поиск** (контекст + SEO под перечисленные запросы) — основной канал
2. **Тендерные площадки** (rostender, b2b-center, zakupki.gov.ru) — реальные лоты есть
3. **Холодные продажи** по списку предприятий, где видно покупки СОЖ через тендеры
4. **chipmaker.ru** — комментарии в обсуждениях, экспертные ответы (не реклама)
5. **Отраслевые выставки** (Металлообработка май/Москва, MITEX ноябрь, региональные)
6. **Email-рассылки** на закупщиков
7. ❌ **НЕ тратить** на YouTube, Telegram-каналы металлообработки, ВК

### Для продукта/сервиса

- **Программа испытаний (PDF на 1–2 страницы)** — что поставляем, что замеряем, сроки 4 недели, итоговый отчёт.
- **«Арсенал технолога»** — комплект, который выдаём с КП: шаблон ТЗ для ОПИ, шаблон отчёта, сравнительная таблица, TCO-калькулятор, презентация для главного инженера.
- **«Карточка контрагента»** — полный пакет для СБ закупщика.
- **Услуга «Сопровождение СОЖ»** — квартальные выезды инженера, замеры, корректировки. Это retention + upsell.

## Стратегическая рекомендация

В первый год **не масштабируйтесь широко**. Сфокусируйтесь на **3–5 «якорных» сделках** в одном-двух сегментах (инструментальное производство + общее машиностроение средних заводов СПб/СЗФО). Получите **глубокие референсы** с конкретными цифрами.

**Не пытайтесь** в первый год:
- Зайти в авиа/оборонку (циклы 12+ месяцев, отраслевые допуски, шансы малы)
- Конкурировать с Лукойлом по цене на массовом сегменте
- Делать массовую рекламу (ROI плохой при отсутствии репутации)

**Сделайте** в первый год:
- Сайт по чеклисту выше
- Контент-машину под SEO (10–15 статей в год)
- Программу пилотных испытаний с готовым «арсеналом»
- 3–5 первых сделок любой ценой
- Накопление обратной связи и валидацию гипотез
