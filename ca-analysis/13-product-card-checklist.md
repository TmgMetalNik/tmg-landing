# Карточка товара СОЖ — что цепляет технолога

> **Что внутри:** конкретный список того, что реально важно технологу/закупщику в карточке товара СОЖ. Не «общие пожелания», а приоритизированный чеклист с привязкой к **тому, что есть/нет у конкурентов** (по итогам сбора TDS-данных с сайтов 5 конкурентов, 10 продуктов).
>
> **Главный инсайт:** конкуренты публикуют **только базовые параметры**. 5 ключевых характеристик НЕ показывает почти никто (ЧШМ, антикоррозия по DIN, биостойкость в неделях, пенообразование по ASTM, заявленный иностранный аналог). **Если TMG это опубликует — обойдёт всех конкурентов на уровне карточки товара.**

---

## ⭐ Главное правило

Технолог открывает карточку товара с **одной мыслью**: «подойдёт мне это или нет, и можно ли с этим работать без сюрпризов».

Он НЕ читает маркетинговый текст. Он ищет **3 вещи**:
1. **Подходит ли продукт под мою задачу** (мой материал × моя операция × мой станок)
2. **Можно ли доверять цифрам** (есть ли скачиваемый TDS, есть ли документы)
3. **Можно ли быстро попробовать** (есть ли образец, как заказать, какая цена)

Вся остальная карточка — это **снятие возражений**, чтобы технолог не закрыл вкладку и не пошёл дальше.

## 🔴 БЛОК 1 — Заголовок и идентификация (первое, что видно)

**Что должно быть:**
- ✅ **Название продукта** жирно и крупно (CutMax EP 620 / AquaMax Cool 68 / AquaLite M 90)
- ✅ **Тип СОЖ** под названием — масляная / водосмешиваемая полусинтетическая / водосмешиваемая минеральная / синтетика
- ✅ **Короткая фраза-хук в одну строку** — «для тяжёлого точения нержавейки и титана» / «универсалка для большинства операций» / «эконом-вариант для типовых задач»
- ✅ **Бейджи-маркеры** (3–4 штуки): «Для нержавейки», «Без хлора», «Биостойкость 9 мес», «Производство РФ»

**Психология:** технолог за 2 секунды понимает «это про меня или нет». Если про него — остаётся. Если нет — закрывает.

## 🔴 БЛОК 2 — Применимость (под мой материал × мою операцию)

Это **самый важный блок** для технолога. Если он не находит свою комбинацию материал/операция — закрывает карточку.

**Что должно быть в виде таблицы или иконок:**

### По операциям:
- Точение ✓ / ✗
- Фрезерование ✓ / ✗
- Сверление обычное ✓ / ✗
- **Глубокое сверление (L/D > 5)** ✓ / ✗
- Нарезание резьбы ✓ / ✗
- Шлифование ✓ / ✗
- **MQL (микроподача)** ✓ / ✗
- Хонингование, развёртывание ✓ / ✗

### По материалам:
- Углеродистые стали ✓ / ✗
- **Нержавейка** (12Х18Н10Т, 08Х18Н10) ✓ / ✗
- **Жаропрочные стали** ✓ / ✗
- Чугун ✓ / ✗
- Алюминий и сплавы ✓ / ✗
- **Алюминий с магнием** ✓ / ✗ (отдельно — не все СОЖ для него подходят)
- Медь, латунь, бронза ✓ / ✗
- **Титан и титановые сплавы** ✓ / ✗

### По режимам:
- Лёгкий / Средний / **Тяжёлый**
- Скорость резания (мм/мин — диапазон)

### По типу станков:
- Универсальный с ЧПУ ✓ / ✗
- Высокоскоростные обрабатывающие центры ✓ / ✗
- Шлифовальные станки ✓ / ✗

**Бенчмарк:** все 10 продуктов конкурентов перечисляют материалы и операции, но **в виде сплошного текста**. Если TMG сделает это **в виде понятной таблицы или иконок-чекбоксов** — резко выделится.

## 🔴 БЛОК 3 — TDS / технические характеристики

Здесь главная развилка по конкурентам:
- **ЛИККОР и AQUACOOL** дают полный TDS с цифрами (плотность, pH, вязкость, концентрация, метод антикоррозии)
- **Техносож и Русские Компоненты** — только маркетинговый текст без цифр

Если TMG будет в первой группе — **войдёт в шорт-лист серьёзных производителей**.

### Базовый минимум (есть у Ликкор и AQUACOOL — обязательно):
| Параметр | Единица | Зачем технологу |
|---|---|---|
| Плотность при 20°C | г/см³ | Для расчёта дозировки и контроля |
| Цвет / внешний вид концентрата | — | Визуальная идентификация партии |
| Цвет / внешний вид рабочей эмульсии | — | «как должно быть» при контроле |
| pH рабочего раствора | — | Контроль активности |
| Концентрация (рекомендованная по операциям) | % | Главное для подбора |
| Коэффициент рефрактометра | — | Контроль концентрации в цехе (рефрактометр — стандартный инструмент) |
| Кинематическая вязкость | мм²/с | Совместимость со станком |
| Срок хранения | месяцы | Закупочное планирование |
| Температура хранения | °C | Логистика и склад |
| Жёсткость воды для разведения | мг-экв/л | Критично — половина проблем СОЖ от плохой воды |

### Премиум-уровень (НЕ публикует почти никто — наш шанс отстройки):
| Параметр | Единица | Зачем технологу | У кого есть |
|---|---|---|---|
| **Нагрузка сваривания (ЧШМ)** | кН | Для тяжёлых операций / EP-классификация | Никто (упомянуто у Ликкор-200 без цифры) |
| **Антикоррозия по DIN 51360-2** | баллы 0–4 | Стандарт, понятный закупщикам ОПК | Никто (Ликкор даёт «стружка-фильтр» — другой метод) |
| **Биостойкость в неделях/месяцах с указанием метода** | недели | Главная боль рынка | Только Ликкор-100 («до 12 мес») |
| **Пенообразование по ASTM D892 / Ross-Miles** | мл / время оседания | Для высоких оборотов и MQL | Никто |
| **Точка вспышки (для масляных)** | °C | Безопасность, страхование | Никто в выборке |

**👉 Если TMG опубликует 5 параметров премиум-уровня — это станет главным маркетинговым отличием.**

## 🔴 БЛОК 4 — «Без чего» (экологические/гигиенические маркеры)

Очень важно для алюминия, авиа, оборонки, пищёвки. Технолог часто **гуглит конкретно «СОЖ без хлора»** или «СОЖ без формальдегида».

**Что писать (если применимо):**
- Без хлора
- Без серы
- Без фосфора
- Без бора
- Без силикона
- Без нитритов
- Без формальдегида / фенольной смолы
- Без вторичных нитросаминов
- Биоразлагаемость (если заявлено)

**Бенчмарк:** Техносож и Русские Компоненты явно перечисляют «без чего». ЛИККОР и AQUACOOL — частично. **Это «обязательная база»**, не отстройка.

## 🔴 БЛОК 5 — Замена иностранному аналогу

🚨 **Никто из 10 проверенных конкурентов прямо не называет иностранный бренд-аналог.** Только общие фразы «голландские/немецкие аналоги».

**Что должно быть на странице каждого продукта TMG:**

> **Замена:** Castrol Hysol XBB, Houghton Hocut 795B, Blaser Vasco 5000, FUCHS Ecocool MK, Quaker Cimstar 35-451

Это:
- Прямой ответ на запрос «замена Castrol Hocut» в Яндексе
- Делает выбор технолога в 10 раз проще («ага, это вместо нашего Houghton — берём на пилот»)
- Закупщик может сразу обосновать выбор перед руководством («это аналог нашему привычному, а не что-то новое»)
- **SEO-магнит** для всех запросов «замена X»

**👉 Это может быть отдельный мини-блок «Заменяет:» с 3–5 логотипами / названиями импортных продуктов.**

## 🔴 БЛОК 6 — Документы (для закупщика)

Закупщик отдельно проходит по странице товара и **смотрит документы**. Если их нет в виде PDF — для серьёзного предприятия товара как бы и нет.

**Что должно быть скачиваемо:**
- ✅ **TDS (Technical Data Sheet)** — паспорт продукта в PDF
- ✅ **Паспорт безопасности (MSDS)** — обязательно для химии
- ✅ **Декларация соответствия** (или сертификат ТР ТС)
- ✅ **ТУ** (Технические условия) — лучше скачиваемая первая страница
- ✅ **Образец паспорта качества партии** — образец того, что приходит с каждой отгрузкой
- ✅ **Рекомендации по применению** — отдельный PDF с подробными режимами
- ✅ **Сертификат ISO 9001** компании (если есть)

**Бенчмарк:** только ЛИККОР делает это в полном объёме (15+ PDF). У AQUACOOL — есть TDS PDF. У Техносожа и Русских Компонентов — **PDF нет вообще**, только маркетинг.

**👉 Если у TMG будут все 7 типов PDF на каждой карточке — это уровень федерального производителя.**

## Главный вывод

**На рынке СОЖ карточка товара уровня «всё, что нужно технологу» — отсутствует.** Никто не делает полный пакет.

Если TMG сделает карточку с 14 блоками выше — она станет **самой полной на рынке**, и это автоматически создаст репутацию «серьёзного производителя» **до того**, как технолог позвонит.

**3 главные отстройки**, которых нет ни у кого:
1. **Заявленный иностранный аналог** на каждой карточке
2. **5 «премиум»-параметров TDS** (ЧШМ, антикоррозия по DIN, биостойкость в неделях, пенообразование, точка вспышки)
3. **Прямой контакт инженера** с фото

Эти 3 пункта — относительно дешёвые в реализации, но **на 10x повышают доверие** на этапе сравнения карточек.

2. **Заполнить параметры по 3 флагманским продуктам TMG** (например, AquaMax Cool 68, CutMax EP 620, AquaLite M 90).
3. **Сверить с этим файлом** — какие из 14 блоков на карточках TMG уже есть, какие пустуют.
4. **Сделать первую карточку** одного товара по полному чеклисту (например, AquaMax Cool 68) — это станет шаблоном для остальных.
5. **Через месяц** — провести аудит первой карточки: что работает, что нет, что доработать.
