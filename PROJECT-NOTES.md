## Как я обычно работаю по этому проекту
1. Правлю файлы локально в **VS Code** (корень — `tmg/`).
2. Деплою изменения на хостинг через **curl по FTPS** (креды беру из `deploy-creds.txt`):
   ```bash
   curl --ssl-reqd -k -T <file> "ftp://server273.hosting.reg.ru/www/theosmg.ru/<file>" --user "<user>:<pass>"
   ```
   Удаление файла — DELE с **абсолютным путём**: `-Q "DELE /www/theosmg.ru/<file>"`
   (pre-quote выполняется до CWD, поэтому относительный путь даёт 550).
3. При необходимости фиксирую в git и делаю **git push** (origin — чистый URL, токен передаю inline разово, в `.git/config` не храню).
