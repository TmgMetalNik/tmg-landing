# Группа 5а. Брендовые конкуренты

**Приоритет:** П2 (тёплый — ищет аналог).
**Типы РК:** только Поиск.
**ЦА:** клиент знаком с конкурентом → ищет аналог / замену / актуальную цену.
**Важно по модерации Яндекса:** **НЕ пишем имя конкурента в тексте объявления и заголовках** (нарушение прав на товарный знак). Используем «импортный аналог», «замена импортной СОЖ», «российский аналог». Имена конкурентов — только в ключах.
