# Карта групп TMG · приоритеты × тип РК

**Логика приоритетов:**
- **П1** — собственники/закупка/завод. Опт, бочки, тендеры, ОЕМ, импортозамещение, инфраструктура цеха. Главная цель ставок.
- **П2** — технологи / гл. инженеры. Выбор продукта под операцию, средний чек.
- **П3** — инфо/проблемы/розница. Только прогрев и ретаргет.

**Гео:** Москва + Санкт-Петербург (старт).
**УТП в основе:** российский производитель, СПб-площадка, инженерное сопровождение, лабораторные испытания, импортозамещение.
