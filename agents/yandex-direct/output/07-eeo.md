# Группа 7. ЭЭО / электроэрозия

**Приоритет:** П2 (узкая инженерная ниша).
**Типы РК:** только Поиск.
**ЦА:** оператор или технолог проволочно-вырезного / прошивного эрозионного станка.
**Замечание:** TMG в основной линейке СОЖ для механической обработки. Для EDM — направление через Taikyuu-X / специальные диэлектрики. **На лендинге должен быть подраздел «Диэлектрики для EDM»**, иначе посадка будет нерелевантна. Если такого раздела нет — Гр.7 в первый запуск не включаем (согласовать с Евгением).
