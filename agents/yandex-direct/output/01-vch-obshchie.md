# Группа 1. ВЧ общие

**Приоритет:** П2 (широкий пул, дорогая семантика).
**Типы РК:** Поиск + РСЯ + МК.
**ЦА:** мик-микс (от собственника до студента-технолога).
**Логика бюджета:** охватом не злоупотребляем — в Поиске только высокочастотники + жёсткие минусовки. РСЯ для дешёвого охвата и ретаргета. МК — для теста авто-аудитории.
