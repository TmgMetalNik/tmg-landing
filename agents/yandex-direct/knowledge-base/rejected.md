# Rejected · что НЕ зашло клиенту (и почему)

**Главное правило:** перед тем как использовать любой поинт — сверить с этим списком. Если есть здесь — НЕ использовать.

## 1. Прямой отказ клиента

| Поинт | Почему отказал | Чем заменили |
|---|---|---|
| **«Образец 1 л бесплатно»** | Притягивает халявщиков и розницу, не отсекает B2B | Выезд инженера + расчёт экономии (НО см. п.2 — «расчёт экономии» тоже под вопросом, лучше «выезд инженера TMG» без обещаний) |
| **«Парк станков»** | Клиент не использует эту формулировку | «Оборудование», «материалы», «любые металлы обработки» |
| **«Сравним вашу СОЖ с нашей на лабораторном стенде»** | Клиент **не сравнивает** свою СОЖ с СОЖ клиента | «Замена импортной марки 1:1 по характеристикам» (без слова «сравнение» и «стенд») |
| **«Перед закупкой — лабораторное сравнение на ваших режимах»** | То же — клиент не сравнивает | Не используем |
| **«−11% потерь трения / снижение износа −11%»** | Клиент сказал «нет» к этой цифре в заголовках | Не используем как явную цифру в заголовках/текстах. (Технически правильная формулировка — «потери трения», а не «износ». Если решат вернуть — только так.) |
| **«11 продуктов в каталоге»** | Жёлтый кандидат → клиент сказал «нет» | Используем общую формулировку «масляные, полусинт., водосмешиваемые» |
| **Имена линеек CutMax / AquaMax / AquaLite в обычных группах** | Клиент сказал «нет» в общих заголовках | В заголовках общих групп — нет. В Гр.5б (брендовые свои) — допустимо. |
| **«+30% ресурса инструмента», «+40% ресурса метчика»** | Я выдумал — нет данных | Не используем |
| **Гео в текстах («МСК», «Москва за 48 ч»)** | Решили писать общефедерально | Гео настраивается как таргет в Директе, в текстах не упоминаем |
| **Иероглиф 耐久 в креативах** | Клиент решил не использовать | Не используем |
| **«До/после в цифрах» как стилистика креатива** | Клиент решил не брать | Не используем как стилистику. (Конкретные «до/после» в схемах допустимы — см. «Карта боли».) |

## 2. Я выдумал — НЕ использовать без подтверждения клиента

| Выдумано | Где использовалось | Что писать вместо |
|---|---|---|
| **«Аудит за 3 дня»** | старые БС, тексты | Просто «Аудит цикла СОЖ» (без срока) или «Сроки поставки 2 дня» (это согласовано — но это про поставку, не про аудит) |
| **«Договор за 1 день»** | старые тексты | Не используем |
| **«Контракт на год с фикс цены»** | старые тексты | Не используем |
| **«−30% к цене импорта»** | старые РСЯ | Не используем |
| **«Цена от 380 ₽/л»** | старые тексты | Не пишем цены без согласования |
| **«Расчёт экономии за смену»** | БС «Выезд инженера» | Заменить на «Выезд инженера TMG · подбор СОЖ в вашем цеху» |
| **«44/223-ФЗ» как УТП** | БС, тексты | Использовать только если клиент подтвердит реальный опыт работы по этим законам |
| **«Лабораторный отчёт о Taikyuu-X»** | старые тексты | Цифру −11% не используем (см. п.1), значит и про отчёт пока не пишем |

## 3. Имена конкурентов в текстах объявлений

**Запрет** на упоминание в заголовках/текстах: Лукойл, ТКС, Blaser, Castrol, Mobil, Shell, Petro-Canada, Motul, Texaco, Fado, ITEO.

- В **ключах** — можно (за этим и приходит трафик).
- В **тексте** — пишем «российский аналог», «замена импортной СОЖ», «импортная марка».
- Причина: модерация Яндекса + риск претензий по ТЗ.

## 4. Старые БС — статус «требует переделки»

В старом универсальном наборе 8 БС часть содержит непринятые формулировки. **Переделать перед использованием:**

| БС (старая) | Проблема | Что делать |
|---|---|---|
| «Аудит цикла СОЖ» / описание «Отчёт за 3 дня» | «3 дня» выдумано | Убрать «3 дня» из описания |
| «Замена импортной СОЖ» / описание «Лабораторный стенд → 1:1» | «Стенд» — клиент не сравнивает | Описание заменить: «Российский аналог 1:1 по характеристикам» |
| «Taikyuu-X · −11%» / описание «Модификатор трения, лаб. отчёт о снижении износа» | −11% не используем + «износ» вместо «потерь трения» | Убрать БС полностью или переписать без цифры |
| «CutMax / AquaMax / AquaLite» как отдельные БС | Имена линеек не используем в общих группах | Для общих групп — заменить на «Масляные СОЖ» / «Водосмешиваемые СОЖ» / «Эконом-линейка». Для Гр.5б — оставить как есть. |
| «Выезд инженера TMG» / описание «Расчёт экономии за смену + аудит на ваших станках» | «Расчёт экономии» — выдумка | Описание: «Подбор СОЖ под ваше оборудование» |

## 5. Тексты, которые НЕ зашли (история)

| Текст | Почему не зашёл |
|---|---|
| «Сравним вашу СОЖ с нашей на стенде лаборатории. Без перенастройки техпроцессов.» | Клиент не сравнивает |
| «Перед закупкой — лабораторное сравнение на ваших режимах. Замена без срыва.» | Клиент не сравнивает |
| Любые тексты с «парк станков» | Клиент эту формулировку не использует |

- ❌ **«耐久 как идентичность»** (стилистика 4)
